# React-intro

A minimal **React + Vite** starter project that sets up React with fast refresh (HMR) and ESLint rules using Vite as the build tool.

##  About

This repository provides a simple starting template for building React applications using **Vite** â€” a fast development build tool with Hot Module Replacement (HMR) support. It includes basic configuration to get React working out of the box with ESLint.  

Perfect for beginners learning React or anyone who wants a lightweight React setup without Create React App.

##  Features

-  React 18+
-  Powered by **Vite** for fast dev environment and optimized production build
-  Basic ESLint configuration
-  Simple folder structure
-  Ready to extend with additional tools and libraries

## Installation

Clone the repository:
   git clone https://github.com/Sapkotasumita/React-intro.git
   cd React-intro
   
Install dependencies:
npm install
# or
yarn

Start the development server:
npm run dev
# or
yarn dev

Open your browser and go to:
http://localhost:3000
